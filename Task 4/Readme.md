# Superstore Sales & Profit Analysis

This repository contains an end-to-end data analysis project on the Superstore dataset, including exploratory data analysis in Python and an interactive Excel dashboard for business insights.

## Project Overview

The project combines data analysis and business intelligence reporting:

1. Python-based exploratory data analysis (EDA) and visualizations.
2. An interactive Excel dashboard for executive-level performance monitoring.

The objective is to analyze sales, profitability, product performance, regional trends, and operational impact.

## Repository Contents

- `Superstore_Dashboard.xlsx` – Interactive Excel dashboard with KPIs, pivot charts, and slicers.
- Codebook / Analysis Notebook – Python-based data cleaning, analysis, and visualizations.

## Python Analysis

The data analysis phase includes:

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Sales and profit trend analysis  
- Regional and category-level performance evaluation  
- Visualizations using Python libraries  

## Excel Dashboard Features

The interactive dashboard includes:

- Total Sales  
- Total Profit  
- Profit Margin (%)  
- Total Orders  
- Loss-Making Products  

Visual Components:

- Monthly Sales and Profit Trend  
- Profit by State Analysis  
- Product Profitability Insights  
- Category Performance Comparison  
- Shipping Mode and Discount Impact  
- Interactive slicers for dynamic filtering  

All visuals update dynamically based on user selections.
